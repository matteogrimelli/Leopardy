Site Information

Leopardy is a game inspired by the quiz show "Jeopardy!". The site allows you to create custom game boards, save your questions, and play in multiplayer mode. Additionally, you can perform fact-checking on the questions using ChatGPT.

Functionalities

  • Board Creation: Create a game board with a maximum of 8 rows and 6 columns (the first column is reserved for points and the others for questions in each category).
  • Question Modal: Each board cell opens a single modal where you can enter the question and answer. If the modal is closed without saving, the inputs are cleared.
  • Board Saving: The game board is saved as a JSON file, where questions and answers are encoded in Base64.
  • Playing the Game: Import the JSON board, select the number of players, and play the game. When you click a board cell, a full-screen overlay displays the question. You can reveal the answer and even request a fact-check using ChatGPT.
  • Scoreboard: A scoreboard at the bottom of the page displays player boxes with inputs for names, total scores, and buttons to adjust scores based on the points of the current question.

Tutorial

How to Create a Board
  1. Go to the Create page (create.html).
  2. Enter the board name and update the category (in the header row) and the point value (in the first cell of the data row).
  3. Use the Add row and Add col buttons to add rows and columns (within the specified limits).
  4. Click on Enter Question in a cell to open the modal, enter the question and answer, and save the data.
  5. Finally, click Save table to download the board as a JSON file.
How to Play
  1. Go to the Play page (play.html).
  2. Enter the number of players and import the previously saved JSON board file.
  3. The board will be displayed along with the scoreboard. Each board cell initially shows "try this" and changes to "show again" once clicked.
  4. Click on a cell to open a full-screen overlay displaying the question.
  5. Click Show Answer to reveal the answer and Fact Check to send the question to ChatGPT for a brief fact-check.
  6. Use the "+" and "-" buttons in the scoreboard to update the players' scores based on the current question's points.
  7. After you have used the initial settings (number of players and JSON file), those elements will disappear to make room for the game.
